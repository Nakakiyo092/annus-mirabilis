# Hardware type

The USB2CANFDV1 has three LEDs.

| Label  | Color |
|--------|-------|
| RDY    | Red   |
| RX     | Green |
| TX     | Blue  |


# LED behavior
The rules are applied from top.

### Startup

* RX & TX: Blink 5 times in turn

### Error stored

* RX & TX: Constant ON

### CAN bus closed

* RX: Blink by USB Rx
* TX: Always ON

### CAN bus open

* RX: Blink by CAN Rx
* TX: Blink by CAN Tx
